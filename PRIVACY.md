# Bulwark privacy policy

Last updated: 2026-07-03

Bulwark is built to protect your privacy, so it collects as little as possible.

## What Bulwark does not do

- It does not collect, store, or transmit your browsing history.
- It has no analytics, no telemetry, and no tracking of any kind.
- It has no account system and no login.
- It does not send your data to us or to any third party. There is no Bulwark server.

## What Bulwark stores, and where

All data stays in your browser:

- Your settings (allowlist, blocked sites, schedule, filter toggles, theme, and custom or user rules) are stored with Chrome's extension storage. If you have Chrome Sync enabled, Chrome, not Bulwark, syncs these settings across your devices.
- Block counts and a short in-memory list of recently blocked requests are kept locally so the popup and dashboard can show statistics. The recent-request list is not persisted and is cleared when the browser restarts.

## Network access

Bulwark makes network requests only to:

- Download the public filter lists it uses to block ads and trackers (for example EasyList and EasyPrivacy), and any custom filter-list URLs you add yourself.

It does not contact any other server.

## Permissions

Bulwark requests broad host access so it can block and hide content on the sites you visit. This access is used solely for blocking and cosmetic filtering on the page you are viewing. Bulwark never reads or exfiltrates page content.
